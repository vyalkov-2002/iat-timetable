# Расписание Ирбитского аграрно-технологического техникума

## Что это?

Это небольшая программа, которая скачивает всё доступное расписание техникума и выкладывает его на сайте в простом и компактном виде.
Такое расписание можно, например, заскриншотить и поставить на обои и экран блокировки телефона.
Таким расписанием удобно делиться в чате группы.

Данные загружаются из [личного кабинета студента](https://lk.ecp.egov66.ru/) автоматически.
Следовательно, при возникновении любых неточностей в личном кабинете расписание будет неправильным и там, и здесь.

Этот проект никак не аффилирован с техникумом, его разрабатывает один студент в свободное от учёбы время.

## См. также
